# 安全政策 / Security Policy

## 支持的版本 / Supported Versions

| 版本号 | 是否受支持         |
| ------ | ------------------ |
| 0.0.x  | :white_check_mark: |

## 报告漏洞 / Reporting a Vulnerability

我们认真对待安全漏洞。如果你发现安全问题，请遵循以下步骤：

We take security vulnerabilities seriously. If you discover a security issue, please follow these steps:

1. **不要公开报告** — 请勿打开公开的 Issue
   **Do NOT open a public issue**

2. 通过 [GitHub Security Advisories](https://github.com/jadepam/mail-agent/security/advisories/new) 报告
   Report via [GitHub Security Advisories](https://github.com/jadepam/mail-agent/security/advisories/new)

3. 请包含：问题描述、复现步骤、影响评估
   Please include: description, reproduction steps, and impact assessment

4. 我们会在 **7 个工作日**内响应
   We aim to respond within 7 business days

## 处理流程 / What to Expect

- 确认收到报告 / Acknowledgment of receipt
- 定期更新修复进度 / Regular updates on fix progress
- 修复发布后在版本说明中致谢（可选）/ Credit in release notes (if desired)

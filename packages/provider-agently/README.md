# @mail-agent/provider-agently

Mail Agent Agently Mail 适配器 — 通过 agently-cli 操作 Agent 原生邮箱。

## 特点

- 对接 Agently Mail 平台，支持 AI Agent 专属邮箱
- agently-cli OAuth 授权
- 实现 `MailProvider` 统一接口

## 安装

```bash
pnpm add @mail-agent/provider-agently
```

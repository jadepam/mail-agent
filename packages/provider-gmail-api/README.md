# @mail-agent/provider-gmail-api

Mail Agent Gmail API 适配器 — 通过 Gmail REST API 收发邮件。

## 特点

- 走 Gmail REST API 而非 IMAP，翻墙友好
- OAuth2 浏览器授权，token 自动刷新
- 完整支持收发、搜索、文件夹管理

## 安装

```bash
pnpm add @mail-agent/provider-gmail-api
```

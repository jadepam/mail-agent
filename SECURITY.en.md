# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security vulnerabilities seriously. If you discover a security issue, please follow these steps:

1. **Do NOT open a public issue** — please report privately
2. Report via [GitHub Security Advisories](https://github.com/jadepam/mail-agent/security/advisories/new)
3. Please include: description, reproduction steps, and impact assessment
4. We aim to respond within **7 business days**

## What to Expect

- Acknowledgment of receipt
- Regular updates on fix progress
- Credit in release notes (if desired)

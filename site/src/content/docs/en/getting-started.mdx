## Agent Quick Setup

Copy the prompt below and send it to your AI Agent's chat window to auto-install and configure.

### Skill Mode

> Recommended. The Agent auto-installs CLI + Skill — no manual steps needed.

```text
Please read https://raw.githubusercontent.com/jadepam/mail-agent/main/cli-setup.md and follow the skill mode steps to install and configure Mail Agent.
```

Works with: Claude Code / Cursor etc.

### MCP Mode

> Requires Mail Agent CLI installed locally first.

```text
Please read https://raw.githubusercontent.com/jadepam/mail-agent/main/cli-setup.md and follow the mcp mode steps to install and configure Mail Agent.
```

Works with: Claude Code / Cursor / Windsurf etc.
